# README

# 背景信息
在数据安全越来越重要的当下，作者的公司（传统制造业公司）突然被客户要求建立ISO27001体系，所以紧急突击了一下，搜集了一些资料，主要是基于ISO27001信息安全、网络安全和隐私保护-信息安全管理系统相关的标准文档（主要是中文+内审检查清单）：
分类如下：
0 表示国家法律
1 表示法规和章程
2 表示和ISO27001匹配的内容
3 表示辅助支撑的内容
4 表示参考内容
